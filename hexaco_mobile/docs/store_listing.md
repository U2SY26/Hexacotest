## Short Subtitle (EN)

## Description (KR)
HEXACO 성격 모델을 기반으로 한 상황형 질문으로
나의 성향을 6가지 요인(H/E/X/A/C/O)으로 분석합니다.

- 과학적으로 검증된 HEXACO 모델 기반
- 60/120/180 문항 중 선택
- 100가지 유형 중 가장 가까운 유형 추천
- 결과는 기기에만 저장 (서버 전송 없음)
- 광고는 화면 하단에서만 노출

※ 본 테스트는 비공식이며 전문 심리 진단을 대체하지 않습니다.

## Description (EN)
Assess your personality across 6 HEXACO factors using situational questions.

- Built on the validated HEXACO model
- Choose 60/120/180 questions
- Match with the closest type among 100
- Results stored locally on device (no server upload)
- Ads appear only at the bottom of screens

This test is unofficial and does not replace professional assessment.
